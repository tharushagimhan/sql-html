This repository features a database system for a computer diploma institute designed to monitor students' academic progress.
The system includes key tables such as student, which tracks student details; diploma, outlining diploma programs; module,
detailing module specifics; department, associating departments with lecturers; lecturer, capturing lecturer information; modulecoordinator, 
defining module coordinators; and pastperformance, recording students' exam results. 
This structured approach enables efficient management of student performance and enhances communication among students, lecturers, and administrative staff.
